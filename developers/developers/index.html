Authenticate as an agent

The API uses OAuth 2.0 (OpenID Connect), client-credentials grant:

  1. Create a client ID & secret.
  2. Set RAPIDATA_CLIENT_ID and RAPIDATA_CLIENT_SECRET in the environment.
  3. Construct RapidataClient() — it fetches and refreshes a bearer token automatically.
  4. Without the SDK: POST https://auth.rapidata.ai/connect/token with grant_type=client_credentials, then send the token as Authorization: Bearer ….
